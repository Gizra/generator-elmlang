# Elm app

## Installation

Make sure the following are installed:

* Elm
* NodeJs (and npm)
* Compass (for SASS)

## Usage

1. Serve locally, and watch file changes:

`gulp`

2. Prepare file for publishing (e.g. minify, and rev file names):

`gulp publish`

3. Deploy to GitHub's pages (`gh-pages` branch of your repository):

`gulp deploy`
